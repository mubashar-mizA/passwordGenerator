# Password Generator Web App

A React-based password generator application styled with Tailwind CSS. It generates secure passwords and evaluates their strength levels, providing suggestions for users based on length and complexity.

## Features

- Generate random passwords
- Assess password strength: `Fair`, `Strong`, `Very Strong`
- User-friendly input fields with padding and placeholder text
- Note on password security, recommending users change their password every two weeks

## Technologies Used

- **React**: For building the interactive user interface
- **Tailwind CSS**: For modern, responsive styling
- **JavaScript**: For password generation and strength evaluation
